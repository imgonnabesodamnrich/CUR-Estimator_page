논문 노트: 제약된 미관측 복구 추정기 (CUR-Estimator)

논문 제목: CUR-Estimator: Towards reliable missing data imputation for aero-engine degradation process

저널: Neurocomputing (Elsevier), Vol. 649, 2025

DOI: 10.1016/j.neucom.2025.130876

1. 연구 배경

항공 엔진과 같은 복잡한 장비의 예후 및 건전성 관리(PHM)에서 센서 데이터의 무결성은 고장 진단 및 수명 예측에 매우 중요합니다. 하지만 실제 운용 환경에서는 센서 고장, 정비 요구 사항, 통신 제한 등으로 인해 데이터 결측(Missing Data)이 빈번하게 발생합니다.

결측 데이터를 단순히 삭제하면 다변량 시계열의 연속성이 파괴되므로 데이터 보정이 필요합니다. 항공 엔진 데이터 보정 작업에는 두 가지 주요 과제가 있습니다:

  1. 불규칙한 샘플링 시간 간격: 비행 임무마다 기간이 다르며 이륙, 순항 등 각 단계의 지속 시간도 다릅니다. 전통적인 보정 방법은 등간격을 가정하므로 시간 경과와 엔진 성능 저하 사이의 실제 관계를 반영하지 못합니다.
  2. 신경망 보정의 이상치 문제: 딥러닝은 특징 추출에 뛰어나지만, 산업 현장의 강한 노이즈나 장기 결측 구간에서 신경망이 물리적 범위를 벗어나는 극단적인 값을 생성하는 경우가 있어 신뢰성이 떨어집니다.

그림 1 신경망 보정 방법은 제약이 부족할 경우 실제 분포에서 벗어난 불합리한 이상치를 생성할 수 있습니다.

2. CUR-Estimator 방법의 핵심 아이디어

이러한 문제를 해결하기 위해 본 논문은 제약된 미관측 복구 추정기 (CUR-Estimator)를 제안합니다.

이 방법의 핵심은 "신경망의 비선형 피팅 능력"과 "전통적인 통계학 방법의 안정성"을 결합하는 것입니다. 설계된 신경망 아키텍처를 통해 복잡한 패턴과 시간 간격의 영향을 캡처하는 동시에, 훈련 단계에서 통계적 보정 알고리즘(PCHIP)을 제약 조건으로 도입하여 출력 범위를 제한함으로써 이상치 발생 가능성을 크게 낮춥니다.

그림 2 CUR-Estimator의 전체 실행 프로세스 및 이중 제약 메커니즘.

3. 모델 주요 아키텍처

CUR-Estimator는 두 가지 핵심 요소가 협력하여 작동합니다:

1. 간격 인식 시간 보정 네트워크 (ITIN)
GRU를 기반으로 개선된 이 모듈은 불규칙한 시간 간격을 처리하기 위해 자가 주의(Self-Attention) 메커니즘을 도입했습니다. 이 메커니즘은 인접한 기록 사이의 시간 차이를 가중치로 변환합니다. 이를 통해 모델은 비행 기록 간격이 짧으면 이력 데이터에 의존하고, 간격이 길면 이력의 영향력을 감쇠시켜 문맥에 맞는 합리적인 보정을 수행합니다.

그림 3 ITIN의 내부 구조 및 시간 감쇠 메커니즘.

2. 통계적 제약 컴포넌트
심각한 결측 구간에서의 이상치를 방지하기 위해 PCHIP(Piecewise Cubic Hermite Interpolating Polynomial)를 통계적 제약으로 사용합니다. 훈련 중 손실 함수에 신경망 출력과 PCHIP 결과의 차이를 반영함으로써, 신경망이 복잡한 분포를 학습하면서도 매끄러운 통계적 범위 내에서 결과를 내도록 강제합니다.

4. 실험 검증 및 성능 결과

C-MAPSS 시뮬레이션 데이터와 실제 항공사 데이터를 사용하여 테스트를 수행했습니다. CSDI, GP-VAE, BRITS 등 최신 모델과 비교한 결과:

  • 오차 성능: 반복 실험 결과, CUR-Estimator는 두 데이터 세트 모두에서 가장 낮은 MSE와 MAE를 기록하여 정밀한 보정 능력을 입증했습니다.
  • 경계 편차: 제약이 없는 기본 모델에 비해 참값과의 편차 범위가 현저히 좁아졌으며, 이는 통계적 제약의 효과를 보여줍니다.
  • 노이즈 강건성: 다양한 SNR 환경에서도 성능이 안정적이었으며, 표준 편차가 비교 방법들에 비해 명확하게 작았습니다.

5. 결론

CUR-Estimator는 시간 간격 정보를 모델 가중치에 인코딩하여 불규칙 샘플링 시계열 모델링 문제를 해결했으며, 통계적 제약을 통해 산업 데이터에서 신경망이 이상치를 출력하는 결함을 개선했습니다. 또한 풍력 발전 데이터 테스트에서도 높은 정확도를 보여주어, 비등간격 및 고잡음 환경의 결측 데이터 처리에 대한 신뢰성 있는 프레임워크를 제공합니다.
